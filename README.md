# ImageFlow 🖼️

A modern React-based image upload application featuring a sleek glass morphism design and intuitive user experience.

## ✨ Features

- 📤 Drag & Drop Image Upload
- 🖼️ Real-time Image Preview
- 📱 Responsive Design
- 🎨 Modern Glass Morphism UI
- ✅ Form Validation
- 🗃️ Gallery View
- 🔄 Multi-image Upload Support (up to 5 images)
- 🎯 Individual Image Management

## 🛠️ Built With

- React + Vite
- React Router DOM
- Zod (Form Validation)
- React Hook Form
- Tailwind CSS

## 🚀 Getting Started

1. Clone the repository
```bash
git clone https://github.com/Dev-Dhruv0/ImageFlow.git
```

2. Install dependencies
```bash
cd ImageFlow
npm install
```

3. Run the development server
```bash
npm run dev
```
